# EHR-Agentic-AI

This repository contains resources for EHR-focused AI workflows, including the `MIMIC-IV-Data-Pipeline` project and local data assets used during development.

## Contents

- `MIMIC-IV-Data-Pipeline/`: data preprocessing and modeling pipeline code for MIMIC-IV.
- `data/`: local datasets and generated artifacts (ignored by Git).

## Quick Start

1. Create and activate a Python environment.
2. Install dependencies from `MIMIC-IV-Data-Pipeline/requirements.txt`.
3. Follow `MIMIC-IV-Data-Pipeline/README.md` for pipeline usage details.
