6. Comparison: Prompt_v2 vs Previous GPT-4o-mini Prompt_v1

The previous GPT-4o-mini results (prompt_v1) are taken from docs/LLM_EXPERIMENT_REPORT_GPT4O_MINI.md (section 10.1/10.2).
Below, Delta = prompt_v2 - prompt_v1 in percentage points.

Approach ACC v1 ACC v2 Delta ACC Sens v1 Sens v2 Delta Sens Spec v1 Spec v2 Delta Spec F1 v1 F1 v2 Delta F1
Zero-Shot 80.82 73.07 -7.75 57.69 83.60 +25.91 89.55 69.10 -20.45 62.24 62.99 +0.75

Comparison Takeaways

  • Prompt_v2 produces a clear sensitivity gain across all approaches, with the largest gain in Zero-Shot+ (+27.09 points).
  • This gain is accompanied by a substantial specificity drop across all approaches (about -14 to -21 points).
  • Net result: ACC decreases in every mode (about -4.93 to -7.75 points), while F1 improves modestly (+0.75 to +1.62 points).
  • Parse health improved in v2: prompt_v1 had 1 unparseable row in three modes, while prompt_v2 has 0 unparseable rows in all four modes.

In short, relative to prompt_v1, prompt_v2 behaves as a more positive-class-seeking configuration: better recall and slightly better F1, but lower specificity and lower accuracy.
